# Next.js 14 Complete Course

Next.js aaj kal ka sabse popular React framework hai — Vercel, TikTok, Twitch jaise companies use karti hain. Is course mein hum **zero se Next.js master** karenge.

## Next.js Kya Hai?

Next.js ek **React framework** hai jo aapki website ko:
- **Fast** banata hai (SSR aur SSG support)
- **SEO-friendly** banata hai (search engines mein top pe aata hai)
- **Production-ready** out of the box deta hai

## Course Outline

Is course mein hum cover karenge:

1. **Setup & Installation** — pehla Next.js project
2. **App Router** — naya routing system samjho
3. **Server vs Client Components** — kab kaunsa use karna hai
4. **Data Fetching** — async/await ka jadu
5. **Routing & Navigation** — dynamic routes
6. **Styling** — Tailwind CSS integration
7. **API Routes** — backend bhi Next.js mein
8. **Deployment** — Vercel pe live karo

## Pehla Step: Project Banao

Terminal mein ye command chalao:

```bash
npx create-next-app@latest my-app
```

Phir options mein TypeScript, Tailwind, App Router select karo. Bas — project ready!
