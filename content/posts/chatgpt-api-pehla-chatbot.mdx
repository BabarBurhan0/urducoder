# ChatGPT API Se Apna Pehla Chatbot Banao

AI dunya badal raha hai aur ChatGPT iska sabse bara example hai. Aaj hum seekhenge ki kaise OpenAI ke ChatGPT API ka istemal karke aap apna **custom chatbot** bana sakte hain — wo bhi sirf Node.js use karke.

## Kya Chahiye Hoga?

Shuru karne se pehle aapke paas ye cheezein honi chahiye:

- **Node.js** installed (v18 ya zyada)
- **OpenAI account** with API key
- **Basic JavaScript knowledge**
- **VS Code** ya koi bhi text editor

## Step 1: OpenAI API Key Lo

Sabse pehle [platform.openai.com](https://platform.openai.com) pe account banao. Phir Settings → API Keys mein jao aur ek nayi key generate karo. Ye key safe rakho — kisi ko mat dena!

## Step 2: Project Setup

Apne terminal mein ye commands chalao:

```bash
mkdir my-chatbot
cd my-chatbot
npm init -y
npm install openai dotenv
```

## Step 3: Code Likho

Project folder mein `chatbot.js` file banao aur ye code likho. Phir `.env` file mein apni API key save karo. Bas — aapka pehla chatbot tayar hai!

## Conclusion

Aaj humne dekha ki ChatGPT API kitna powerful hai. Aap is se chatbots, content generators, translators, aur bohat kuch bana sakte ho.

**Happy Coding!** 🚀
